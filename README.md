# Breathe ESG Dashboard

A full-stack ESG emissions analytics dashboard built using:

- Django REST Framework
- React.js
- Tailwind CSS
- SQL Server
- Pandas

## Features

- CSV Upload
- ESG Dashboard Analytics
- Approval Workflow
- Suspicious Record Detection
- Filters
- Sidebar Navigation
- Real-time Dashboard Updates
